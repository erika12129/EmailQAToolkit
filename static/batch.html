How to Use Batch Processing

  1. Upload a requirements JSON file that contains the expected sender, subject, preheader, and UTM parameters.
  2. Upload multiple email HTML files to validate against the requirements.
  3. Click "Run Batch QA" to process all emails.
  4. View the results for each email, which will be displayed separately.

Requirements JSON Format:

{
  "sender": "marketing@company.com",
  "subject": "Special Discount - 25% Off",
  "preheader": "Limited time offer: 25% discount on all products!",
  "utm_parameters": {
    "utm_source": "email",
    "utm_medium": "marketing",
    "utm_campaign": "special_offer"
  }
}
